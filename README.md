# BoxDesign AI Mobile App

AI-powered packaging design assistant.

## Tech Stack
- **Frontend**: React Native (Expo)
- **State**: Zustand
- **Backend**: FastAPI (Python)
- **AI**: Stable Diffusion, ESRGAN, GPT-4

## Setup
1. `npm install`
2. Create `.env` from `.env.example`
3. Run `npx expo start`
